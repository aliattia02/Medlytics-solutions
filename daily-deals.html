Problem & Solution

The Problem

Egyptian shoppers struggle with scattered promotional information across multiple stores, paper flyers that get lost, and no easy way to compare offers across different supermarkets. Price-conscious families waste time visiting multiple stores or miss out on deals entirely.

Our Solution

Daily Deals aggregates all supermarket catalogues in one app with smart filtering by location, real-time offer tracking, and a unified shopping basket. Users can browse offers offline, save favorites, and never miss a deal - all with full Arabic RTL support tailored for Egyptian shopping habits.

System Architecture

1. User Action & State Management

User interactions trigger Redux dispatches, updating the global state with Redux Toolkit slices for catalogues, offers, basket, and favorites.

2. Middleware & Synchronization

Custom middleware intercepts actions to sync data with Firebase for authenticated users while maintaining local state consistency.

3. Firebase Integration

Firestore handles real-time data sync, Authentication manages user sessions, and Cloud Storage serves catalogue images and PDFs via global CDN.

4. Multi-Layer Caching

Memory cache ‚Üí AsyncStorage ‚Üí Firebase with smart invalidation (30min-1day TTL). Full offline read access with background sync when reconnected.

5. UI Rendering

React Native components re-render based on state changes, with optimized image loading, lazy loading, and RTL support via I18nManager.
